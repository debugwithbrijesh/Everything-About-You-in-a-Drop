## Description
This project analyzes blood test datasets and presents results via interactive dashboards built with Streamlit. It highlights abnormal values with color coding and provides visualizations such as line plots, scatter plots, histograms, box plots, pie charts and heatmaps.

## Project Structure
- `app.py` - Streamlit dashboard / main app
- `blood_dataset.csv` - Sample dataset (CSV)
- `project_report.docx` - Project synopsis / report
- `dashboard_screenshots/` - Example dashboard screenshots
- `requirements.txt` - Python packages required
